# Progression Drafts

Черновик для идей о прогрессии: технологиях, частях башен, зданиях и стенах. Записи можно оставлять в сыром виде, а позже раскладывать по игровым системам.

## Raw Ideas

- Первое здание: укрытие. Грубый навес из дерева и веток. Даёт людей. Позволяет исследовать собирательство.
- Первая пушка не имеет ствола: только запускатель и корзина с камнями.
- Собирательство открывает грубый ствол из найденного старого полого ствола и здание "пункт сбора хлама".
- Пункт сбора хлама и укрытие можно объединить в "Дом сталкера" (название под вопросом).
- Сталкер ходит по разным местам разрушенного войной мира и собирает разные вещи, травы, животных.
- Дом сталкера открывает сбор семян.
- Дом сталкера также открывает инструменты из хлама.
- Инструменты из хлама открывают сарай с инструментами.
- Сарай с инструментами и домик сталкера вместе образуют дом лесоруба.
- Дом лесоруба открывает деловое дерево.
- Деловое дерево позволяет строить хороший деревянный ствол и деревянный дом.
- Деревянный дом даёт больше людей в upkeep.
- Сбор семян открывает огород с дикими растениями.
- Огород объединяется с домиком сталкера в дом травника.
- Дом травника открывает растениеведение.
- Растениеведение открывает отравленные камни как боеприпас. Они отравляют, но требуют осторожного обращения и немного снижают скорость стрельбы.
- Растениеведение открывает здание "Поле".
- Поле увеличивает производство людей и биомассы у соседних зданий на 25%.
- В целом прибавки и увеличения должны быть значимыми, примерно в районе 25%.
- Поле с деревянным домом вместе образует ферму.
- Ферма открывает технологию деньги, здание рынок и улучшение "тренированный стрелок".
- Тренированный стрелок относится куда-то в район зарядного устройства пушки и увеличивает скорострельность.
- Это первый апогей, который требует не только людей, но и золота.
- Рынок открывает технологию "Мистицизм". Тематика: появились гадалки и похожие персонажи.
- Мистицизм открывает здание "Дольмен".
- Дольмен вместе с домиком сталкера открывает "Хижину шамана".
- Хижина шамана открывает технологию "Волшебные камни".
- Волшебные камни открывают парящую платформу для башни.
- Парящая платформа быстрее поворачивается. В ней волшебные камни.
- Хороший дом вместе с сараем с инструментами открывает мастерскую.
- Мастерская открывает технологии работа по дереву и работа по камню.

## Technologies

- Собирательство
  - Открывается укрытием.
  - Открывает грубый ствол.
  - Открывает пункт сбора хлама.
- Сбор семян
  - Открывается домом сталкера.
  - Открывает огород с дикими растениями.
- Инструменты из хлама
  - Открываются домом сталкера.
  - Открывают сарай с инструментами.
- Деловое дерево
  - Открывается домом лесоруба.
  - Открывает хороший деревянный ствол.
  - Открывает деревянный дом.
- Растениеведение
  - Открывается домом травника.
  - Открывает отравленные камни.
  - Открывает поле.
- Деньги
  - Открываются фермой.
  - Первый апогей, требующий людей и золото.
- Тренированный стрелок
  - Открывается фермой.
  - Улучшение в районе зарядного устройства пушки.
  - Увеличивает скорострельность.
  - Первый апогей, требующий людей и золото.
- Мистицизм
  - Открывается рынком.
  - Тематика: гадалки, предсказатели, ранние мистические практики.
  - Открывает дольмен.
- Волшебные камни
  - Открываются хижиной шамана.

  - Открывается мастерской.
- Работа по камню
  - Открывается мастерской.

## Tower Parts

- Запускатель
  - Стартовый компонент первой пушки.
  - Работает без ствола.
- Корзина с камнями
  - Стартовый боезапас первой пушки.
- Грубый ствол
  - Открывается собирательством.
  - Сделан из найденного старого полого ствола.
- Хороший деревянный ствол
  - Открывается деловым деревом.
  - Более качественная деревянная версия ствола после появления лесорубной цепочки.
- Парящая платформа
  - Открывается волшебными камнями.
  - Платформа для башни.
  - Быстрее поворачивается.
  - Использует волшебные камни.
- Зарядное устройство / механизм заряжания
  - Возможное место для улучшения "тренированный стрелок".
  - Улучшение увеличивает скорострельность.
- Отравленные камни
  - Открываются растениеведением.
  - Боеприпас с эффектом отравления.
  - Требуют осторожного обращения.
  - Немного снижают скорость стрельбы.

## Conversion Notes

- Проверить название "Дом сталкера": оно хорошо передаёт постапокалиптического сборщика, но может звучать слишком современно или отсылочно.
- Возможная ранняя цепочка: укрытие -> собирательство -> пункт сбора хлама + грубый ствол -> дом сталкера -> сбор семян -> огород -> дом травника -> растениеведение -> отравленные камни.
- Вторая ветка от дома сталкера: дом сталкера -> инструменты из хлама -> сарай с инструментами -> дом лесоруба -> деловое дерево -> хороший деревянный ствол + деревянный дом.
- Уточнить термин "деловое дерево": возможно, имелось в виду "деловое дерево" как пригодная древесина/строевой лес или отдельная технология обработки дерева.
- Значимые бонусы должны быть ощутимыми: ориентир около 25%, а не мелкие проценты.
- Ферма выглядит как первый крупный экономический апогей: вводит золото/деньги, рынок и первое дорогое боевое улучшение.
- Возможная фермерская цепочка: дом травника -> растениеведение -> поле -> ферма -> деньги + рынок + тренированный стрелок.
- Возможная мистическая цепочка: рынок -> мистицизм -> дольмен -> хижина шамана -> волшебные камни -> парящая платформа.
- Возможная ремесленная цепочка: деревянный дом + сарай с инструментами -> мастерская -> работа по дереву + работа по камню.
- Термины закреплены: "деньги" - технология, "рынок" - здание.
- Отравленные камни можно оформить как боеприпас с плюсом к урону по времени и штрафом к скорости стрельбы.
